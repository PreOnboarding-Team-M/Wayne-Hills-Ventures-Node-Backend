# PreOnboarding-Team-M-Wayne-Hills-Ventures-Node-Backend

## 사용 기술스택

### JavaScript

### Express

### sequelize

### mysqlDB

## 실행방법

### 1.해당 레포지토리를 clone합니다.

```shell
git clone https://github.com/Choi-jeonghoon/Wayne-Hills-Ventures-Node-Backend
```

### 2. 다운 받으신 폴더로 들어갑니다

```shell
cd Wayne-Hills-Ventures-Node-Backend
```

### 3. 의존성들을 설치합니다.

```shell
npm i
```

### 4. 실행시킵니다!

```shell
npm start
```
